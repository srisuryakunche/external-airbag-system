# 🚗 External Airbag Safety System

This project simulates a smart external airbag system using Arduino and an ultrasonic sensor.

## 🔧 Components
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- LEDs (Safe, Warning, Critical)
- Buzzer
- Logic Analyzer

## ⚙️ Working
- Measures distance using ultrasonic sensor
- Indicates safety level using LEDs
- Activates buzzer and airbag in critical condition

## 📊 Waveform
TRIG and ECHO signals analyzed using logic analyzer.

## ▶️ Simulation
Run in Wokwi using sketch.ino and diagram.json
